# Executive AI Advisory Institute (EAAI)

A premium platform for AI transformation advisory, training, certification, and measurement — built for executive decision-makers.

## Tech Stack

- **Framework:** SvelteKit (Svelte 5 with runes)
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4 + custom CSS design system
- **Deployment:** Vercel adapter

## Features

- 🎯 **Executive Advisory** — Strategic AI transformation engagement models
- 📚 **Course Catalog** — Self-paced learning with filtering and categories
- 🏆 **Certification Paths** — Foundation through Executive-level certifications
- 🎓 **Live Training** — Expert-led workshops and cohort programs
- 👔 **Leadership Programs** — Tailored programs for C-suite and department leaders
- 🔓 **Open Source** — Community frameworks, templates, and tools
- 📅 **Bookings** — Strategy call scheduling
- 🛒 **Store** — Digital products and resources

## Design

- Dark-mode first with custom CSS variable design system
- Cinematic shooting star hero animations
- Scroll-triggered reveal animations (IntersectionObserver)
- Custom branded 404 error page
- Fully responsive, mobile-first layouts

## Getting Started

```sh
npm install
npm run dev -- --port 6969
```

## Build

```sh
npm run build
npm run preview
```
